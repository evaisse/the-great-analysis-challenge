# Imba Chess Engine

A high-performance chess engine implementation in **Imba 2**, compiling to Node.js.

## Features
- Polyglot-compliant CLI protocol
- Full move generation (castling, en passant, promotion)
- Minimax AI with Alpha-Beta pruning
- Fast perft implementation

## Development
- `make build`: Compiles Imba to JS
- `make docker-build`: Builds the Docker container
- `make docker-test`: Runs a quick integration test in Docker
